使用說明

如何開始使用 DRAssistance？

首次使用需要進行以下步驟：

  1. 下載並安裝 DRAssistance 應用程式
  2. 使用 Google/Apple 帳號註冊登入
  3. 應用程式會自動初始化加密系統
  4. 開始添加病人資料或使用語音功能建立任務

如何添加和管理病人資料？

病人管理非常簡單：

  • 點擊右上角的 ➕ 按鈕手動添加病人
  • 填寫基本資訊（姓名、床號、年齡等）
  • 敏感資訊（姓名、病歷號）會自動加密存儲
  • 使用下拉選單快速切換不同病人
  • 滑動病人卡片可進行編輯或刪除

如何使用語音功能建立任務？

語音轉任務功能步驟：

  1. 在病人詳情頁面點擊 🎤 麥克風按鈕
  2. 允許應用程式使用麥克風權限
  3. 清楚說出您的任務內容（支援繁體中文）
  4. AI 會自動分析並生成結構化任務
  5. 確認任務內容後保存到病人檔案

如何跨設備同步資料？

跨設備同步完全自動化：

  • 使用相同的 Google/Apple 帳號登入
  • 應用程式會自動從雲端同步加密金鑰
  • 所有病人資料和任務會自動解密顯示
  • 如遇同步問題，可在設定頁面重新同步密鑰

常見問題

我的資料有多安全？

DRAssistance 採用醫療級 AES-256-GCM 加密技術，僅對患者敏感識別資訊（姓名、病歷號、住院號）進行加密。每位使用者擁有獨立的加密金鑰，確保資料的絕對安全。

忘記帳號或無法登入怎麼辦？

DRAssistance 使用 Firebase Authentication，您可以直接使用 Google 或 Apple 帳號登入，無需記住額外密碼。所有加密金鑰都與您的 Google/Apple 帳號綁定，確保跨設備安全同步。

語音識別不準確怎麼辦？

請確保：1) 在安靜的環境中使用；2) 清楚地說出醫療術語；3) 檢查麥克風權限是否開啟；4) 語音識別目前針對繁體中文優化，建議使用標準國語發音。

可以離線使用嗎？

目前 DRAssistance 需要網路連線才能正常運作，因為語音轉任務功能使用雲端 AI 服務，資料同步也依賴 Firebase。未來版本將考慮加入離線功能。

如何備份我的資料？

您的資料會自動備份到 Firebase 雲端，登入相同帳號即可在任何設備存取。如需額外備份，可在設定頁面使用密鑰同步功能確保資料安全。

系統需求

📱 iOS

  • iOS 12.0 或更高版本
  • 支援 iPhone 和 iPad
  • 需要麥克風和網路存取權限
  • 原生語音辨識 (SFSpeechRecognizer)
  • 支援繁體中文語音輸入

📱 iPad

  • iPadOS 13.0 或更高版本
  • 支援所有 iPad 機型
  • 大螢幕優化界面
  • 分割畫面多工支援
  • Apple Pencil 相容性

🌐 Web (PWA)

  • 現代瀏覽器 (Chrome, Safari, Edge)
  • 支援 JavaScript 和 WebAssembly
  • 麥克風存取權限
  • 穩定的網路連線
  • 支援漸進式網頁應用安裝
