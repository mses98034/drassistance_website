使用說明

如何開始使用 Doctor Assistant？

首次使用需要進行以下步驟：

  1. 下載並安裝 Doctor Assistant 應用程式
  2. 註冊帳號登入
  3. 應用程式會自動初始化加密系統
  4. 開始添加病人資料或在iPad上使用Apple Pencil記錄查房重點

如何添加和管理病人資料？

病人管理非常簡單：

  • 點擊右上角的 ➕ 按鈕手動添加病人
  • 填寫基本資訊（姓名、床號、年齡等）
  • 敏感資訊（姓名、病歷號等）會自動加密存儲
  • 使用下拉選單快速切換不同病人
  • 滑動病人卡片可進行編輯或刪除

如何使用Apple Pencil記錄查房重點？

iPad版本支援Apple Pencil手寫功能：

  1. 在病人詳情頁面點擊手寫記錄區域
  2. 使用Apple Pencil直接在螢幕上書寫
  3. 支援自然手寫輸入，無需特殊字體
  4. 手寫內容會即時保存到病人檔案
  5. 可配合其他功能一起使用，提升查房效率

如何跨設備同步資料？

跨設備同步完全自動化：

  • 使用相同的帳號登入
  • 應用程式會自動從雲端同步加密金鑰
  • 所有病人資料、任務和手寫記錄會自動解密同步顯示
  • 如遇同步問題，可在設定頁面重新同步密鑰

常見問題

我的資料有多安全？

Doctor Assistant 採用醫療級 AES-256-GCM 加密技術，對患者敏感識別資訊（姓名、病歷號、住院號）進行加密。每位使用者擁有獨立的加密金鑰，確保資料的絕對安全。

忘記帳號或無法登入怎麼辦？

Doctor Assistant 使用 Firebase Authentication，您可以直接於登入頁面點擊「忘記密碼」進行密碼重設。使用帳號登入。

Apple Pencil手寫功能使用上有什麼要注意的？

請確保：1) 使用原廠或相容的Apple Pencil 2) iPad已更新到支援的系統版本 3) 手寫區域有足夠的空間 4) 手寫內容會即時同步到雲端，確保網路連線穩定。

可以離線使用嗎？

目前 Doctor Assistant 需要網路連線才能正常運作，主要用於資料同步和雲端備份。手寫記錄功能可在離線狀態下使用，但需要網路連線才能同步到其他設備。

如何使用主治醫師分類管理功能？

新的主治醫師管理功能特別適合住院醫師使用。您可以在病人列表中依主治醫師篩選，快速找到特定醫師的病人。此功能讓同時照顧多位主治醫師病人的住院醫師能更有效地管理工作。

顏色標記系統如何使用？

全新的四色標記系統（紅、黃、藍、無標記）讓您可以個人化分類病人。搭配智能篩選功能，可以組合主治醫師和顏色標記進行快速篩選，大幅提升工作效率。

如何編輯個人資料？

在設定頁面中，您現在可以編輯個人資料，包括姓名、職稱（主治醫師/住院醫師/實習醫師）、醫師章號。這些資料會與註冊時的資訊保持同步，但登入郵箱基於安全考量無法修改。

如何備份我的資料？

您的資料會自動備份到 Firebase 雲端，登入相同帳號即可在任何設備存取。如需額外備份，可在設定頁面使用密鑰同步功能確保資料安全。

系統需求

📱 iOS

  • iOS 16.6 或更高版本
  • 目前僅支援 iPhone 和 iPad
  • 需要麥克風和網路存取權限
  • Apple Pencil 完整支援
  • 手寫辨識與即時儲存

📱 iPad

  • iPadOS 16.6 或更高版本
  • 支援所有 iPad 機型
  • 大螢幕優化界面
  • 分割畫面多工支援
  • Apple Pencil 相容性

🌐 Web (PWA)

  • 現代瀏覽器 (Chrome, Safari, Edge)
  • 支援 JavaScript 和 WebAssembly
  • 麥克風存取權限
  • 穩定的網路連線
  • 支援漸進式網頁應用安裝
